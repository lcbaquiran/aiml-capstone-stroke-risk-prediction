# Stroke Risk Prediction (Machine Learning Capstone)

## Overview
This project develops a machine learning model to predict stroke risk using demographic and clinical features.

## Objectives
- Identify key risk factors for stroke
- Compare multiple machine learning models
- Evaluate performance using recall, precision, and AUC
- Assess ethical implications and bias

## Dataset
Stroke Prediction Dataset from Kaggle  
(Soriano, n.d.)

## Methods
- Data cleaning and preprocessing
- Feature engineering and encoding
- SMOTE for class imbalance
- Models: Logistic Regression, Random Forest, SVM
- Evaluation: Accuracy, Recall, Precision, F1, ROC AUC

## Key Findings
- Logistic Regression achieved highest recall
- Random Forest achieved highest accuracy but low recall
- Stroke prediction depends on multiple interacting features

## Ethical Considerations
- Risk of false negatives in healthcare context
- Class imbalance impacts fairness
- Model should be used as decision support, not diagnosis
